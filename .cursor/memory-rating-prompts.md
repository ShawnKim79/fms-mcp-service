- 향후 작업 전반에 폭넓게 적용되는 고수준·구체적·실행 가능한 선호나 규칙
- 명확한 워크플로 단계, 기술 선택, 코딩 스타일 선호
- 이 레포에서 반복 참조되는 실행/테스트/엔드포인트/환경 키 정보

- 이미 `.cursor/agent-rules.md`에 저장소 전역 규칙으로 중복되는 내용
- 개인 로컬 환경, 임시 경로/로그

## 출력 언어
- 설명은 영어로 작성하세요.
